-**Por qué es conveniente incluirlo ?**
es conveniente incluirlo porque asi no se suben al repositorio archivos 
que no son importantes en el proyecto

-**cuando se debe hacer?**
es más recomendable hacerlo cuando se trabaja en grupo , ya que habra
diferencia de sistemas y archivos en cada persona, entonces solo se suben
los archivos necesarios 


-**como configurar el archivo .gitignore?**
la configuracion es simple , se crea el archivo .gitigore 
dentro de el se describe que achivos o archivos con ciertas extensiones
no se subiran
